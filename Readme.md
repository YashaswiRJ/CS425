# CRC Implementation in C++

This project implements Cyclic Redundancy Check (CRC) in C++.

---

## Compilation

### macOS / Linux

1. Navigate to the project directory:

```bash
cd path/to/project
```

2. Compile the program:

```bash
g++ -std=c++17 main.cpp -o main
```

This will create an executable file named `main`.

---

## Run the Program

```bash
./main
```
